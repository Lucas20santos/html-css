# Como funcionam a HTML e as CSS

"Eu desenvolvo em HTML e CSS"

O HTML é uma linguagem focado em conteudo
    - texto
    - imagens
    - videos
    - tabels
O CSS é focado em estilo
    - cores
    - sombras
    - tamanhos
    - posicionamento
O JavaScript é focado em interações
    - menus 
    - animações
    - popup
    - validações
